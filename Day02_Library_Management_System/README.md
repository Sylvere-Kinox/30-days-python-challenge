# Système de gestion de bibliothèque — Day 02

Petit projet Python réalisé pour le challenge "30 Days Python".

## Description

Ce programme simule un système simple de gestion d'une bibliothèque en console. Il permet de gérer un catalogue de livres et d'effectuer des opérations basiques (ajout, suppression, recherche, affichage).

## Fonctionnalités

- Lister les livres disponibles
- Ajouter un nouveau livre
- Supprimer un livre
- Rechercher un livre par titre ou auteur
- Sauvegarde légère en mémoire (pas de base de données)

## Prérequis

- Python 3.8+ (idéalement 3.10/3.11)

## Installation

1. Cloner le dépôt ou télécharger cet exercice.
2. Placer-vous dans le dossier du projet pour Day 2 :

```bash
cd 30-days-python-challenge/Day02_Library_Management_System
```

3. Lancer le script principal :

```bash
python main.py
```

## Utilisation

Le script s'exécute en console et affiche un menu interactif. Suivez les instructions à l'écran pour ajouter, supprimer ou rechercher des livres.
